# React Static Website

This is a simple static React JS project created to understand how React works.

## Purpose
- Learn React folder structure
- Understand components and JSX
- Practice props and state
- Learn basic styling

## Tech Stack
- React JS
- JavaScript (ES6)
- HTML
- CSS

## Features
- Static UI
- Reusable components
- Responsive layout

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Status
This project is for learning purposes only (no backend or API).
